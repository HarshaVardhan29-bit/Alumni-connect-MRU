# AlumniAI — Manav Rachna University

Full-stack alumni networking platform with AI-powered mentorship matching.

## Stack
- **Frontend**: React + Vite (port 5173)
- **Backend**: Node.js + Express (port 5000)
- **Database**: MongoDB

## Setup & Run

### Prerequisites
- Node.js v18+
- MongoDB running locally (`mongod`)

### Backend
```bash
cd alumni-network
npm install
npm run dev
```

### Frontend
```bash
cd alumni-network/frontend
npm install
npm run dev
```

Open http://localhost:5173
